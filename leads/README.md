# Lead archive

This folder is reserved for lead files generated by the Cloudflare Worker.

Expected format:

```text
leads/YYYY-MM-DD/YYYY-MM-DDTHH-MM-SS-sssZ-event_id.json
```

Each generated JSON file contains the original lead payload, archive status,
and downstream delivery metadata when available.

Important: this repository is public. Do not enable public lead archiving unless
you intentionally want personal lead data committed to a public GitHub repo.
